# 3D Model Assets

## Expected files per invention:

### telephone/
- hero.glb - Full assembled telephone model
- diaphragm.glb, housing.glb, coil.glb, receiver.glb, battery.glb

### iphone/
- hero.glb - Full assembled iPhone model
- display.glb, logic-board.glb, battery.glb, camera.glb, speaker.glb

### steam-engine/
- hero.glb - Full assembled steam engine model
- piston.glb, cylinder.glb, flywheel.glb, beam.glb, condenser.glb, boiler.glb

### telescope/
- hero.glb - Full assembled telescope model
- objective-lens.glb, eyepiece-lens.glb, outer-barrel.glb, inner-barrel.glb, mount-rings.glb

## Audio intros (public/audio/intros/)
- telephone-intro.mp3, iphone-intro.mp3, steam-engine-intro.mp3, telescope-intro.mp3
